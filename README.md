# GPM (German Pronounce & Meaning)

GPM is a **fun, offline Chrome extension** I built to help myself learn German words while reading websites.  

Whenever you highlight a German word, GPM instantly tells the English meaning and speaks it a loud  **all offline**! No data is sent or received. Everything runs locally in your browser.

## Features

- Lightweight and private
