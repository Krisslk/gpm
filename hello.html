GPM (German Pronounce & Meaning)

Offline Tool:

This is a fun project and an open source tool to help you learn German words instantly!
No data is sent or received everything runs offline in your browser.

You can update the dictionary anytime by adding words to the local dictionary JSON file.

Sorry for any missing words you can always check the GitHub repo for updates and contributions:
